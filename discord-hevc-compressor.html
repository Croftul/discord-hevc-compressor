Best settings

  • For short gameplay clips, 25 MB with Fast preset is a good first try.
  • If the result still looks too soft, raise the upload target or switch to Medium.
  • For meme clips or screen recordings, turning audio off can save a lot of space.
  • If a clip is still too large, trim it first or reduce frame rate to 30 fps in code.

Reality check

  • HEVC can shrink files well, but browser-side encoding is heavy and may be slow on weak laptops.
  • Very large videos can run into browser memory limits. This works best for short Discord clips.
  • If you want maximum compatibility, add an H.264 mode later as a fallback.
  • You can host this as a static site because everything runs client-side.
